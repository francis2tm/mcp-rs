# mcp-rs

A simple Rust library that provides addition functionality.

## Installation

Add this to your `Cargo.toml`:

```toml
[dependencies]
mcp-rs = "0.1.0"
```

## Usage

```rust
use mcp_rs::add;

fn main() {
    let result = add(5, 3);
    println!("5 + 3 = {}", result);
}
```
